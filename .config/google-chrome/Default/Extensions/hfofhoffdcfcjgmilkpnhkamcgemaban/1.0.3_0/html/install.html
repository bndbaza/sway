«Скачать музыку» - скачивай музыку быстро, удобно и бесплатно!

Политика использования расширения

  • Расширение позволяет скачивать аудиотреки и целые альбомы с сайта Я.Музыка.
  • По правилам Яндекс Музыки для прослушивания и скачивания музыки требуется войти в аккаунт Яндекса.
    Без входа все треки скачиваются по 30 секунд.
  • По умолчанию аудио треки скачиваются с битрейтом 192 kbps.
    При наличии подписки «Плюс» треки скачиваются с битрейтом 320 kbps.
  • Расширение скачивает файлы со всеми метаданными (обложка, название, артисты, альбом, год выпуска и т.д.) только в том случае
    если они имеются на сервере.
  • Расширение предоставляет возможность массового скачивания треков.
    Мы никак не ограничиваем данный функционал, но ограничения могут накладывать ваш браузер и сам сервис, где хранится музыка.
    Если вы ставите на закачку сразу большое количество треков мы не можем гарантировать что все они будут полностью скачены.
    Для скачивания одновременно всех треков необходимо прокрутить страницу с аудио в самый низ.
  • При первом запуске для включения меню расширения в верхнем углу вашего браузера нажмите на значок «Пазл» → «Закрепить расширение».
  • В расширение можно указать собственную папку для скачивания аудиозаписей.
    Опция доступна через меню настроек (правый клик мыши по иконке → Параметры).
    Если в вашем браузере установлена опция «Всегда указывать место для скачивания» - браузер будет показывать соответствующее окно вне зависимости от того указана папка или нет.
    По умолчанию, все аудиозаписи скачиваются в папку, указанную для загрузок всех файлов в вашем браузере.
  • Расширение работает корректно только в браузере Google Chrome. Работа в других браузерах, включая Яндекс Браузер, не гарантируется.
  • Для корректной работы расширения отключите другие приложения для скачивания музыки, видео и т.д.
  • Жалобы по всем вышеуказанным пунктам не принимаются.

Продолжая использовать расширение Вы соглашаетесь со всеми вышеуказанными пунктами.
